# 呼吸器専門医試験 問題集 PWA

## 内容
- 一問一答: 63問
- 5択問題: 16問
- 元ファイル: 総論I_形態機能病態生理_問題集.docx

## ローカルで確認する方法
このフォルダで以下を実行します。

```bash
python -m http.server 8080
```

ブラウザで `http://localhost:8080` を開きます。

## スマホで使う方法
### 簡単な方法
1. GitHub Pages / Netlify / Vercel などにこのフォルダをアップロード
2. スマホのブラウザでURLを開く
3. iPhone: 共有 → ホーム画面に追加
4. Android: メニュー → アプリをインストール または ホーム画面に追加

## 問題を追加する方法
`data/questions.json` に同じ形式で項目を追加します。
一問一答は `type: "qa"`、5択は `type: "mcq"` にします。

## 注意
- 成績は端末のブラウザ内 localStorage に保存されます。
- 別端末同期は未実装です。
