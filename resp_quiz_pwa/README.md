# 呼吸器専門医 問題集 PWA

## 更新内容

- 「一周する」または 10/20/30/50問・任意の問題数を選んで開始できます。
- 設定した問題数が終わると、回答数・正解数・間違い数・正答率を表示します。
- 結果画面から「今回間違った問題だけ」を復習できます。
- 累積で間違えた問題は、モード「累積で間違えた問題」から復習できます。

## GitHub Pagesで差し替えるファイル

既存リポジトリでは、ZIP内の中身をすべて上書きしてください。最低限は以下を差し替えます。

- index.html
- app.js
- style.css
- service-worker.js
- data/questions.json
- data/questions.csv

古い表示のままの場合は、ブラウザ再読み込み、またはスマホのホーム画面アプリを一度削除して再追加してください。
