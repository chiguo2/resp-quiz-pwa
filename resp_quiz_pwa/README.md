## GitHub Pagesでの更新

ZIPを展開し、リポジトリ直下に中身を上書きアップロードしてください。
最低限の差し替え対象は以下です。

- index.html
- app.js
- style.css
- service-worker.js
- data/questions.json
- data/questions.csv

反映されない場合は、ブラウザ再読み込み、またはホーム画面アプリを削除して再追加してください。
