## 教科書参照リンク

各問題の解答表示後に「教科書の該当解説を開く」が表示されます。リンク先では、添付された教科書PDFから対応ページを抽出した閲覧用画像を表示します。PDF本体はPWAに同梱せず、PDFファイルへの直接リンクやダウンロードボタンも設置していません。

参照ページ画像は初回表示時に端末へキャッシュされ、その後はオフラインでも再表示できます。


## v10 教科書参照画面の戻る操作
教科書ページはアプリ内モーダルで開きます。上部の「← 問題に戻る」またはAndroidの戻る操作で、回答中の問題に戻れます。
